POST
https://kaiqiuwang.cc/xcx/public/index.php/api/trainer/lists
参数（Form-Data）：
Key=type; Value=0
Key=city; Value=杭州市
Key=lat; Value=30.20840072631836
Key=lng; Value=120.2120132446289
Key=page; Value=1
Key=sort; Value=0
